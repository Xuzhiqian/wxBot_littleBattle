基本操作
1.join--加入游戏
2.u(dis),d(dis),l(dis),r(dis)--分别向上下左右移动dis距离，玩家不可重叠，不可越界，移动成功后私发位置消息
3.a+@某人--平A某人（必须在攻击范围内，冷却时间内无效）
4.s--获取攻击范围内的视野（冷却时间内无效),并显示当前状态
5.k+@某人--使用英雄技能（冷却时间内无效，如果为非指向性技能则at视为无效）


特性：
移动操作会耗蓝，且耗蓝量随移动距离的增大而增大（二次函数）
每个英雄都有一个被动技能和主动技能
每个英雄每5秒回蓝2点，除了隐者没有回血
攻击范围为周围8个格子
初始时每个玩家的职业通过私发信息来通知，群里大家互相不知道彼此的职业
英雄普通攻击CD:5秒，冷却时间内攻击会倒扣5血
